# Rocket Intelli

IntelliSense extension for Razor/CSHTML files with custom token support.

## Features

- Custom IntelliSense for Razor/CSHTML files
- Support for special Rocket framework tokens
- Auto-completion triggered by @ character

## Development

1. Install dependencies: `npm install`
2. Compile the extension: `npm run compile`
3. Press F5 to open a new VSCode window with the extension loaded
4. Test the extension with .cshtml or .razor files

## Requirements

- Visual Studio Code 1.80.0 or higher
- Node.js and npm

## Extension Settings

This extension contributes the following settings:

* `rocketIntelli.enable`: Enable/disable Rocket Intelli
